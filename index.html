Win10如何搭建自己的hexo博客

准备

从node.js官网中下载node.js，下稳定版（也就是左边的版本）。
下载好之后按照提示一路next直到安装完成就好，但选择存储地址时默认为c盘，记得修改。
！无法显示

开始部署

这一步真的绕了很多弯路。

打开命令行窗口，按以下指令输入

注意，全部指令都要用英文键盘输入
注意，全部指令都要用英文键盘输入
注意，全部指令都要用英文键盘输入
node -v node版本
npm -v npm版本
npm install -g cnmp --registry=https://registry.npm.taobao.org 配置淘宝镜像源
cnpm install -g hexo-cli 安装hexo工具
依次输入以上指令之后，查看版本安装成功与否：hexo -v
一般来说前面几步不会出现任何问题。接下来开始配置blog。
首先选择一个你喜欢的盘，创建一个名为blog的空文档，比如说你创建在D盘，那么按以下指令进行操作。

cd blog
